# Button-Football

足球比赛的狂欢又开始了，喜欢足球的少年看过来哦！游戏里，共有两个队，用按钮代表球员，每个队可以选择自己按钮的颜色与样式，看哪个队能在两分钟之内将最多的球射进对方的球门，就可以获得高分。大家共同期待吧！最大的赢家会是你吗？
